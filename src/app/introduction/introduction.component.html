Introduction

This document is part of the Performance and Architecture Review for the ION USD Bonds sub-Platform installed at HSBC in London (Global Rates Platform).

This Platform Review identifies possible inefficiencies and proposes actions to be taken to optimize the configuration.

ION Consulting will follow up for the planning and implementation of the recommendations provided in this document.

This analysis has been done for the following criteria:

  • Platform topology
  • Failover configuration
  • Server and component performance
  • Routing infrastructure performance

    • The previous review was conducted in June 2016. Where appropriate, references have been provided to observations and recommendations made during the previous review.
