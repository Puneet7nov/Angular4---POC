Analysis constraints and limitations

During this review, we encountered limitations that affected the completeness of the analysis:

  • The CPU statistics are inaccurate for a few servers because hyper-threading (HT) was enabled on them.
  • ION did not receive complete Hardware specifications for a few servers.
  • Servers dedicated to host the custom components do not host a Daemon Slave (DS). As a result, the complete server load analysis for these servers could not be conducted.
  • HSBC did not provide the logs of both the PXE instances running on the sub-Platform.
